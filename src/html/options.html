Token Setting

Please click the extension icon to set up your user token (to access your private repos).

User token can be created in https://github.com/settings/tokens

Some display tips

  • Content showing in the reference blank is for:
    • issue title
    • pull request title
    • release title
  • Content showing in the sub title is for:
    • issue & issue comment body
    • pull request & pull request review comment body
    • commit comment body
    • release body
